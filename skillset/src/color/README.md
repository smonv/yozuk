# yozuk-skill-color

Color converter.

## Examples

### CSS notations

```
#fff
=> 
#ffffff
rgb(1 1 1)
hsl(0 0% 100%)
hwb(0 100% 0%)
hsv(0 0% 100%)
```

```
hsl(235, 100%, 50%, .5)
=> 
#0015ff80
rgb(0 0.083333254 1 / 0.5)
hsl(235 100% 50% / 0.5)
hwb(235 0% 0% / 0.5)
hsv(235 100% 100% / 0.5)
```

```
ForestGreen
=> 
#228b22
rgb(0.13333334 0.54509807 0.13333334)
hsl(120 60.693645% 33.92157%)
hwb(120 13.333332% 45.490192%)
hsv(120 75.53957% 54.509808%)
```
